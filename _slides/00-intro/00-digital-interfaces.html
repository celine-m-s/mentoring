--- layout: slide --- # Concepts --- ## Interface -- ## Interface A point where two systems, subjects, organizations, etc. meet and interact. *(Oxford Dictionary)* --- ## Digital -- ## Digital Involving or relating to the use of computer technology: the digital revolution. *(Oxford Dictionary)* --- ## Digital Interface -- ## Digital Interface Device or program enabling a user to communicate with a computer. --- ## Mobile application (mobile app) -- ## Mobile application (mobile app) Application meant to be run on a mobile device (smartphone, tablet).
[Languages](http://en.wikipedia.org/wiki/Mobile_application_development): Java, C, C++ (Android), Objective-C (iOS), HTML, CSS, Javascript --- ## Server -- ## Server Servers are computer programs running to serve the requests of other programs, the clients (your browsers). Servers accept requests from the client and delivers some data, like webpages. --- ## Software -- ## Software Application that can be run on a computer and performs specific operations. --- ## Hardware -- ## Hardware The machines, wiring, and other physical components of a computer or other electronic system. --- ## Website -- ## Website Web pages served from a single web domain. You can access it using the Internet.
Two different websites: static and dynamic ones. --- ## Web application (web app) -- ## Web application (web app) Application meant to be run on a browser (like Firefox or Chrome). Information displayed to the user depends on its action.
Languages: Ruby, Python, PHP, Javascript, HTML, CSS. --- ## User Interface (UI) -- ## User Interface (UI) This is how your user interacts with your application.
  • Does he know where to click?
  • Does he see how to do that action?
  • ...
