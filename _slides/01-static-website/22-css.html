--- layout: slide --- # CSS Cascading Style Sheets Basics --- ## Linking stylesheet to HTML page ```html ``` --- ## CSS vocabulary ![]({% asset_path 22-css/CSS-voc.png %}) -- ## Example ```css h1 { color: purple; background-color: yellow; } ``` --- ## Colors -- ###Colors (1) ```css body { color: orange; } ``` -- ### Colors (2) ```css body { color: #FFA500; /* hexadecimal color code */ } ``` -- ## Background color ```css body { background-color: black; } ``` -- ## Background image ```css section { height: 300px; background: url(background-jpg) no-repeat center center; background-size: cover; } ``` --- ##Fonts -- ### Font-family ```css h1 { font-family: courier, sans-serif; } ``` -- ### Font size and line height ```css body { font-size: 14px; /* minimum if you want your user to be happy */ line-height: 18px; /* minimum 30% higher than your font-size */ } ``` -- ### Font color ```css body { color: red; } ``` -- ### Underline ```css a { text-decoration: underline; } ``` -- ### Center an inline element ```css p { text-align: center; /* this will center the content, not the paragraph in itself */ } ``` -- ### Center a block element ```css header { width: 800px; margin: auto; } ``` --- ## Sizes How do we create a square of 190px? -- ## Sizes ```css article { height: 190px; width: 190px; } ``` -- ## Radius And how do we create a circle? -- ## Radius ```css img { border-radius: 50%; height: 50px; width: 50px; } ``` --- ##Your turn!
